# Archive

This directory keeps historical reports and generated artifacts that are useful for provenance but too noisy for the repository root.

- `legacy-reports/` contains previous audit/status/profile drafts. Some files intentionally preserve historical pass/fail snapshots and may contradict the current README.
- `generated-artifacts/` contains one-off generated outputs that are not part of the supported runtime path.

Current reviewer-facing documentation lives in `README.md` and `docs/`.
